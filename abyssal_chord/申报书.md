## 项目简介

深渊协奏是一款以 MoonBit 为主要实现语言的回合制卡牌对战游戏引擎。玩家扮演"调音师"，利用声波武器、护甲系统和永久能力与畸变体战斗。项目包含 5,266 行 MoonBit 核心引擎代码，覆盖卡牌战斗、敌人 AI、状态管理、多人对战等完整游戏系统，并提供基于 Next.js 的 Web 可视化前端。

## 项目方向与适用场景

- **方向**：游戏引擎 / 卡牌策略游戏 / 应用生态
- **场景**：单机卡牌对战、局域网多人对战、AI 难度挑战、生存模式
- **复用价值**：MoonBit 游戏开发参考案例，可复用的回合制战斗框架

## 拟实现的核心功能

### 已完成功能
1. **卡牌系统** — 55 张卡牌，4 种流派（低频堡垒、过载冲击、高频刺客、回声幻影）
2. **战斗引擎** — HP/护甲/AP 管理、5 级污染度系统、声爆 Debuff 叠加引爆
3. **敌人 AI** — 7 种敌人（含 2 Boss 二阶段），骰子驱动行动选择，4 级难度
4. **永久能力系统** — 频率锚定、低频共振、痛觉回响、终末定音等 6 种永久能力
5. **遗物与阵型** — 4 个角色专属遗物，5 种战术阵型
6. **游戏模式** — 快速对战、生存模式（无尽波次）、Boss Rush、训练模式
7. **多人联机** — 基于 WebSocket 的局域网双人对战
8. **辅助系统** — 卡牌数据库查询、自定义套牌构建、成就系统（40 项）、战斗回放、AI vs AI 模拟器
9. **CI/CD** — GitHub Actions 自动检查/构建/测试
10. **测试覆盖** — 108 个自动化测试（全部通过）

## 是否为原创项目

本项目为**原创项目**。卡牌体系、数值模型、污染度系统、声爆机制均为自主设计。MoonBit 引擎代码为全新编写，TypeScript 前端基于 Next.js 生态组件构建。

## 技术路线

- **核心引擎**：MoonBit 语言，利用其类型系统、模式匹配、枚举等特性实现复杂游戏逻辑
- **AI 工具使用**：Claude Code 辅助代码生成、Bug 修复、测试用例编写、卡牌平衡分析
- **前端展示**：Next.js + React + Framer Motion，WebSocket 多人实时同步
- **测试策略**：108 个黑盒测试覆盖污染度、卡牌数据、战斗状态、伤害结算、胜利/失败判定
- **CI/CD**：GitHub Actions 自动运行 `moon check` + `moon build` + `moon test`
