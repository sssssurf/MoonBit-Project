\section{项目简介}

深渊协奏是一款以 MoonBit 为主要实现语言的回合制卡牌对战游戏引擎。玩家扮演"调音师"，利用声波武器、护甲系统和永久能力与畸变体战斗。项目包含 \textbf{5,266 行 MoonBit 核心引擎代码}，覆盖卡牌战斗、敌人 AI、状态管理、多人对战等完整游戏系统，并提供基于 Next.js 的 Web 可视化前端。

\section{项目方向与适用场景}

\begin{itemize}[leftmargin=*]
  \item \textbf{方向}：游戏引擎 / 卡牌策略游戏 / 应用生态
  \item \textbf{场景}：单机卡牌对战、局域网多人对战、AI 难度挑战、生存模式
  \item \textbf{复用价值}：MoonBit 游戏开发参考案例，可复用的回合制战斗框架
\end{itemize}

\section{拟实现的核心功能}

\subsection{已完成功能}

\begin{enumerate}[leftmargin=*,nosep]
  \item \textbf{卡牌系统} — 55 张卡牌，4 种流派（低频堡垒、过载冲击、高频刺客、回声幻影）
  \item \textbf{战斗引擎} — HP/护甲/AP 管理、5 级污染度系统、声爆 Debuff 叠加引爆
  \item \textbf{敌人 AI} — 7 种敌人（含 2 Boss 二阶段），骰子驱动行动选择，4 级难度
  \item \textbf{永久能力系统} — 频率锚定、低频共振、痛觉回响、终末定音等 6 种永久能力
  \item \textbf{遗物与阵型} — 4 个角色专属遗物，5 种战术阵型
  \item \textbf{游戏模式} — 快速对战、生存模式（无尽波次）、Boss Rush、训练模式
  \item \textbf{多人联机} — 基于 WebSocket 的局域网双人对战
  \item \textbf{辅助系统} — 卡牌数据库查询、自定义套牌构建、成就系统（40 项）、战斗回放、AI vs AI 模拟器
  \item \textbf{CI/CD} — GitHub Actions 自动检查/构建/测试
  \item \textbf{测试覆盖} — 108 个自动化测试（全部通过）
\end{enumerate}

\begin{itemize}[leftmargin=*]

  \item \textbf{CI/CD}：GitHub Actions 自动运行 \texttt{moon check + moon build + moon test}
\end{itemize}
